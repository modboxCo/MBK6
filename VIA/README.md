# VIA Configuration

Place your VIA JSON configuration files here.

## Usage

1. Download the `.json` file for your MBK6 macropad
2. Place it in this folder
3. Open VIA and load the configuration file via **Settings → Load Device JSON File**

## Files

- `mbk6.json` - Main VIA configuration file

## Notes

- Keep a backup of your VIA configuration file version controlled
- The JSON file defines key mappings, layers, and custom functions
